## 🧠 About the Site

My portfolio is a minimalist, dark-themed, animated single-page app built to showcase my front-end skills and personality.

### 🧩 Tech Stack

- ⚛️ **React.js / Next.js**
- 🎨 **SASS CSS** (or custom CSS)
- 🚀 **Deployed with Vercel**
- 🧠 Possibly enhanced with:
  - `Framer Motion` or CSS animations
  - `React Icons` / SVG components

## 🎨 Color Palette

| Element     | Color        | Description                     |
|-------------|--------------|---------------------------------|
| Background  | `#000000`    | Clean dark mode base            |
| Text        | `#FFFFFF`    | High contrast and clean         |
